supported software
this page uses technology called WebRTC. Current development team has limited it's effort to support only the most convenient. In order to get this working you'd need latest of one of modern web browsers.
WebRTC compatible browsers are supported.
Surely there are many applications and sites that allow you to chat over the net so why bother with new one. Well the goal of this project is not only to have convenient means for communication without any unnecessary registration, ready for you 24/7, but also absolute and full control over it. Either you may trust the fact that open source won't do you no harm or become part of the project yourself!
